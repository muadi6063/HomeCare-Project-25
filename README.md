# HomeCare Appointment Management System

This project consists of a backend and frontend. The backend is developed using .NET and the frontend is based on React.

## Directory structure
The .NET backend is located in the "HomecareApp" folder.
The React frontend is located in the "frontend" folder.

## How to Run the Application
To run the backend, navigate to the "HomecareApp" folder. And use the commands:

dotnet build
dotnet run

This will start the backend on localhost:5011.

To run the frontend, navigate to the "frontend" folder. And use the command:

npm run dev

This will start the backend on localhost:3000/
